# Glossary

Verifier packet: scoped verifier asset with origin, scope, procedure, certification condition, boundaries, residuals, update profile, and circulation status.

Residual: typed record of unresolved, untranslated, unexcluded, missing, preserved, redacted, migrated, or conflicting material.

Residual ledger: append-only event trace for residual add, merge, retire, quarantine, and redact operations.

Conformance profile: core, operational, or federated implementation level.

Authority decision: deny-by-default policy record for an action.

Aperture: capacity for question forms, residual types, translation channels, counter-packets, schema revision, and self-verification.

Overclosure: local verifier gain plus inadequate residual, channel, schema, question-form, or reachability preservation.
